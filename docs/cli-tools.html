CLI Tools & Vim Plugin

Command-line tools and editor integrations for seamless code sharing

Overview

JJ Pastebin provides two powerful command-line tools to integrate code sharing into your development workflow:

Python CLI Tool

A standalone command-line interface for creating, managing, and sharing pastes directly from your terminal.

Vim Plugin

A comprehensive Vim plugin that integrates seamlessly with your editor for instant code sharing.

Quick Start: Both tools support authentication, multiple paste formats, and automatic language detection.

Python CLI Tool

Installation

Install the CLI tool using pip from the project directory:

# Install from project directory
cd jjs-pastebin
pip install -e .

# Or install system-wide (optional)
sudo pip install -e .
sudo ln -sf $(which jj) /usr/local/bin/jj
Note: The CLI tool requires Python 3.8+ and the requests library.

Commands

jj paste [file]

Create a new paste from a file or stdin

Supports: file input, stdin, language detection
jj list

List your recent pastes

Shows: paste ID, title, language, creation date
jj get [paste-id]

Retrieve a specific paste

Options: raw output, syntax highlighting
jj delete [paste-id]

Delete a paste (requires authentication)

Note: Only paste owners can delete

Examples

Basic File Upload
# Upload a Python file
jj paste script.py

# Upload with custom title
jj paste script.py --title "My Script"

# Create private paste
jj paste script.py --private
Pipe Input
# Pipe command output
ls -la | jj paste -

# Pipe with language specification
echo "console.log('hello');" | jj paste - --language javascript

# Pipe with title
cat ~/.bashrc | jj paste - --title "My Bash Config"
Authentication
# Set credentials (one-time setup)
jj config --server http://localhost:8000 --username myuser --password mypass

# Or use environment variables
export JJ_SERVER=http://localhost:8000
export JJ_USERNAME=myuser
export JJ_PASSWORD=mypass
Paste Management
# List your pastes
jj list

# Get paste content
jj get abc123

# Get raw paste (no formatting)
jj get abc123 --raw

# Delete a paste
jj delete abc123

Vim Plugin

Installation

Install the Vim plugin using the provided installation script:

# Navigate to the vim-plugin directory
cd jjs-pastebin/vim-plugin

# Run the installation script
./install.sh

# Or install manually
cp jj-pastebin.vim ~/.vim/plugin/
cp jj-pastebin.txt ~/.vim/doc/
vim -c 'helptags ~/.vim/doc' -c 'quit'
Requirements: Vim 7.0+ with Python support, curl, and clipboard utility (pbcopy/xclip/xsel).

Commands

:JJ

Create a public paste from current buffer or selection

Usage: :JJ (whole buffer), :'<,'>JJ (selection), :.JJ (current line)
:JJPub

Create a public paste (explicit)

Same as :JJ but explicitly public
:JJPriv

Create a private paste

Requires authentication
:JJPaste

Generic paste command with options

Most flexible command with full option support
:JJConfig

Configure server settings

Set server URL, username, and password

Configuration

Add these settings to your ~/.vimrc to customize the plugin:

" Server configuration
let g:jj_pastebin_server = 'http://localhost:8000'
let g:jj_pastebin_username = 'your-username'
let g:jj_pastebin_password = 'your-password'

" Default settings
let g:jj_pastebin_public = 1        " 1 for public, 0 for private
let g:jj_pastebin_copy_url = 1      " Copy URL to clipboard
let g:jj_pastebin_open_browser = 0  " Open paste in browser

" Custom keybindings
nnoremap <leader>jp :JJ<CR>
vnoremap <leader>jp :JJ<CR>
nnoremap <leader>jpr :JJPriv<CR>
vnoremap <leader>jpr :JJPriv<CR>
Tip: You can also configure settings interactively using :JJConfig.

Examples

Basic Usage
" Paste entire buffer
:JJ

" Paste current line
:.JJ

" Paste selection (visual mode)
:'<,'>JJ

" Paste lines 10-20
:10,20JJ
Advanced Usage
" Create private paste
:JJPriv

" Create paste with custom title
:JJPaste title="My Custom Title"

" Create paste with specific language
:JJPaste language="python"

" Create private paste with title
:JJPaste title="Secret Code" private=1
Configuration
" Configure server settings
:JJConfig

" Check current configuration
:echo g:jj_pastebin_server
:echo g:jj_pastebin_username
Language Support

The plugin automatically detects language based on Vim's filetype. Supported languages include:

  • Python → python
  • JavaScript → javascript
  • HTML → html
  • CSS → css
  • Bash → bash
  • JSON → json
  • C/C++ → c/cpp
  • Java → java
  • PHP → php
  • Ruby → ruby
  • Go → go
  • And 30+ more...

Troubleshooting

Common Issues

Ensure the CLI tool is properly installed:

# Check if jj is in your PATH
which jj

# If not found, reinstall
pip install -e .

# Or add to PATH manually
export PATH=$PATH:~/.local/bin

Check your authentication settings:

" Check current settings
:echo g:jj_pastebin_server
:echo g:jj_pastebin_username

" Reconfigure if needed
:JJConfig

" Or set manually in .vimrc
let g:jj_pastebin_username = 'correct-username'

Install clipboard utilities for your system:

# macOS (usually pre-installed)
# pbcopy is available by default

# Linux (X11)
sudo apt-get install xclip
# or
sudo apt-get install xsel

# Linux (Wayland)
sudo apt-get install wl-clipboard
