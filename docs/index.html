Installation

Get up and running in minutes

Docker (Recommended)

git clone https://github.com/jjasghar/jjs-pastebin.git
cd jjs-pastebin
docker-compose up -d

# Access at http://localhost:8000

Local Installation

git clone https://github.com/jjasghar/jjs-pastebin.git
cd jjs-pastebin
pip install -r requirements.txt
python run.py

# Access at http://localhost:5000

CLI Tool Installation

# Install the CLI tool
pip install -e .

# Use the CLI
jj paste myfile.py
echo "hello world" | jj paste -

# Install Vim plugin
cd vim-plugin
./install.sh

Quick Examples

See JJ Pastebin in action

Web Interface

# Create a paste via web interface
def hello_world():
    print("Hello, World!")
    return "success"

if __name__ == "__main__":
    hello_world()

Simply paste your code, select the language, and share the URL!

API Usage

# Create paste via API
curl -X POST http://localhost:8000/api/pastes \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Hello World",
    "content": "print(\"Hello, World!\")",
    "language": "python"
  }'

Perfect for automation and integration with other tools.
